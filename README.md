# site-official-archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容与多个不同来源相关，不针对任何特定域名或网站。

## 项目简介

本仓库是一个静态页面集合，旨在保存和展示一组独立的 HTML 文件。这些文件可能包含信息性内容、历史记录或其他文档形式，其来源和用途各不相同。仓库本身不提供任何动态服务或交互功能，仅作为页面文件的存储与展示平台。

## 目录结构

```
.
├── index.html          # 仓库入口页面，提供页面索引
├── pages/              # 存放所有独立 HTML 页面
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
└── assets/             # 静态资源文件（如样式、脚本、图片）
    ├── css/
    ├── js/
    └── images/
```

- `index.html`：仓库主页，列出所有归档页面的链接。
- `pages/`：每个 HTML 文件代表一个独立归档页面，文件名可反映其内容或编号。
- `assets/`：存放页面引用的样式表、脚本和图片等资源，保持页面结构清晰。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖外部服务或数据库。
- 页面内容可能包含文本、图像、链接等元素，但均以本地或相对路径引用资源。
- 归档页面仅作保存与展示之用，不保证内容的时效性、准确性或完整性。
- 页面标题、文件名和内容结构可能根据归档需求进行适当整理。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `index.html` 或 `pages/` 下的任意 HTML 文件，即可在浏览器中查看。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后在线访问。
